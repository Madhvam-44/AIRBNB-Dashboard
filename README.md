# Airbnb Global Performance Dashboard
 
An interactive Power BI dashboard designed to analyze Airbnb listings across multiple cities and years. The dashboard focuses on understanding platform growth, property distribution, host activity, and customer engagement through data-driven visualizations.



2. Short Description 

The Airbnb Global Performance Dashboard is an interactive Power BI report created to explore and analyze Airbnb listings, hosts, cities, and customer reviews. It helps uncover trends in platform growth, property types, and user engagement across multiple years, enabling better understanding of Airbnb’s market expansion and performance.


3. Tech Stack

The dashboard was built using the following tools and technologies:

• 📊 **Power BI Desktop** – Primary data visualization tool used to design the interactive dashboard.
• 📂 **Power Query** – Used for data cleaning, transformation, and preparing the dataset for analysis.
• 🧠 **DAX (Data Analysis Expressions)** – Used for creating calculated measures, KPIs, and dynamic metrics.
• 📝 **Data Modeling** – Relationships created between datasets to enable filtering and meaningful insights.
• 📁 **File Format** – .pbix for development and .png for dashboard preview images.


4. Data Source

The dataset used in this project contains information about Airbnb listings across multiple cities.

The dataset includes details such as:

* Listing information and property types
* Host data and number of listings per host
* City-level listing distribution
* Customer reviews and engagement metrics
* Historical listing trends from **2008–2021**

This dataset was used to explore patterns in Airbnb’s platform growth, property availability, and customer interaction.

5. Features / Highlights

 • Business Problem

Airbnb operates globally with thousands of listings, hosts, and customer interactions. Analyzing such large datasets manually makes it difficult to identify meaningful trends in listings, property types, and market growth.

Important questions such as:

* How has Airbnb grown over the years?
* Which property types dominate the platform?
* How are listings distributed across cities?
* How do external events impact platform activity?

…are difficult to answer without a structured analytical tool.

• Goal of the Dashboard

The goal of this dashboard is to create a **visual analytics tool** that:

* Helps explore Airbnb listing data across multiple cities.
* Identifies growth trends in the Airbnb platform over time.
* Analyzes property type distribution and host participation.
* Provides insights into customer engagement through reviews.


 Walkthrough of Key Visuals

Key KPIs (Top Section)
The dashboard highlights important platform metrics including:

* Total Listings: 279,712
* Total Hosts: 182,024
* Cities Covered: 10
* Property Types: 144
* Total Reviews: 5.3M+

These KPIs provide a quick overview of Airbnb’s platform scale and activity.

Year-wise Listing Growth (Line Chart)
A time-series visualization showing the growth of Airbnb listings from **2008 to 2021**, helping identify periods of rapid expansion and slowdown.

Airbnb Lifecycle Visualization
A visual representation of Airbnb’s lifecycle phases including:

* Introduction
* Growth
* Maturity
* Decline
* Reinvention

This helps interpret how the platform evolved over time.

Property Type Distribution (Comparison Chart)
Compares major property categories such as:

* Entire Place
* Private Room
* Hotel Room
* Shared Room

This visualization highlights which accommodation types dominate the platform.

• Business Impact & Insights

**Platform Growth Analysis**
The dashboard reveals how Airbnb expanded rapidly during its early growth stage and scaled globally over time.

**Property Market Insights**
Entire homes and private rooms represent the majority of listings, indicating user preference for private accommodation.

**Customer Engagement Trends**
Review data highlights strong customer participation and engagement on the platform.

**External Event Impact**
Major global events such as **COVID-19** impacted listing trends and temporarily slowed platform growth.

**Data-Driven Understanding**
The dashboard enables analysts, researchers, and business strategists to better understand Airbnb’s market dynamics and platform performance.
